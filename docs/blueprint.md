# **App Name**: Design Showcase

## Core Features:

- Landing Page: A visually appealing landing page to introduce your design skills.
- About Me Section: An 'About Me' section with a creative photo placement.
- Skills and Experience: Showcase design skills, experience, and software knowledge.
- Portfolio Display: Display the portfolio with a bold 'My Works' heading.
- Category Filtering: Filtering options for portfolio items across categories: branding posters, gaming content, social media posters, movie posters, brand designs, logos, and t-shirt designs.
- Contact Form: A minimal contact form for potential clients.
- Contact Form Submission: Submit contact form to a specific database that you will select later.

## Style Guidelines:

- Primary color: Blood red (#8B0000) for emphasis and contrast in a dark design.
- Background color: Ivory black (#292421) as the dark theme base.
- Accent color: A slightly lighter desaturated red, like brick (#B22222), to draw the eye while maintaining the dark atmosphere.
- Body and headline font: 'Clash Grotesk' (sans-serif) for all text elements to maintain a minimal design. Note: currently only Google Fonts are supported.
- Minimal layout to ensure the focus remains on the showcased designs.
- Subtle transitions on hover for portfolio items to create an interactive experience.
