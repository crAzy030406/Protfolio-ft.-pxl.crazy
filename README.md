# crazy.pxl - Graphic Designer Portfolio

Welcome to the personal portfolio website of **crazy.pxl**, an experienced graphic designer with over 5 years in the industry. This repository contains the source code for a modern, animated portfolio built to beautifully showcase creative works, skills, and professional experience.

## ✨ Features

- **Dark Mode First & Custom Effects:** A sleek, forced dark theme with aesthetic `GlowingBalls` background effects and custom typography (`Inter` and `Unbounded` fonts).
- **Smooth Animations:** Integrated with `framer-motion` for elegant viewport scroll animations across all sections (Hero, Skills, Agency, Contact).
- **Modern UI Components:** Accessible and beautiful components built with Radix UI primitives (`shadcn/ui`) and styled using Tailwind CSS.
- **AI & Integrations:** Set up with Google Genkit AI and Firebase for advanced backend capabilities and AI-driven features.
- **Responsive Layout:** fully optimized for mobile, tablet, and desktop viewing.

## 🛠️ Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Library:** React 18
- **Styling:** Tailwind CSS, `tailwindcss-animate`
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **UI Components:** [Radix UI](https://www.radix-ui.com/)
- **Backend & AI:** Firebase, Google Genkit AI, Next.js Turbopack

## 🏗️ Project Structure

- `src/app/` - Next.js App Router configurations and main layout.
- `src/components/` - Reusable UI components, sections, effects, and layout elements.
- `src/ai/` - Genkit AI configuration scripts.
- `src/firebase/` - Firebase initialization and config.
- `src/hooks/` & `src/lib/` - Custom React hooks and utility functions.

## 🚀 Getting Started

To run this project locally, you will need Node.js installed on your machine.

1. **Clone the repository:**
   ```bash
   git clone <your-repo-url>
   cd Protfolio-ft.-pxl.crazy-main
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```
   *The development server runs with Turbopack on port 9002 (`http://localhost:9002`).*

4. **Build for production:**
   ```bash
   npm run build
   npm start
   ```
